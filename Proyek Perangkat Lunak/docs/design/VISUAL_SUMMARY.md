## 🎬 QUICK START (10 MIN)

### STEP 1: Install Plugin
```
Figma.com
  → Resources
  → Community → Plugins
  → Search "Figma Tokens"
  → Install
```
⏱️ 2 minutes

### STEP 2: Setup File
```
New File
  → Name: "Task Planner - Design System"
  → Create
```
⏱️ 1 minute

### STEP 3: Open Plugin
```
Toolbar → Plugins → "Figma Tokens"
  → Plugin panel opens
```
⏱️ 1 minute

### STEP 4: Get JSON
```
Open: docs/design/FIGMA_TOKENS.json
  → Ctrl+A (select all)
  → Ctrl+C (copy)
```
⏱️ 1 minute

### STEP 5: Paste JSON
```
Plugin text area
  → Ctrl+V (paste)
  → JSON appears
```
⏱️ 1 minute

### STEP 6: Apply Tokens ✨ MAGIC HAPPENS!
```
Click: "Apply Tokens" button
  → Wait 1-2 seconds
  → Check Assets panel
  → Colors! Typography! Spacing!
  → ✅ 50+ tokens ready!
```
⏱️ 1 minute

### STEP 7: Verify
```
Create rectangle test
  → Apply color from Assets
  → ✅ It works!

Create text test
  → Apply typography from Assets
  → ✅ It works!
```
⏱️ 2 minutes
