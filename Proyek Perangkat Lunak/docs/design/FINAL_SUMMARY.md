## 📊 COMPARISON: PlantUML vs Figma Tokens

### Draw.io + PlantUML
```
Keuntungan:
✅ Built-in, tidak perlu plugin
✅ Cepat untuk wireframe simple

Kelemahan:
❌ Limited design tokens (hanya ~20)
❌ Static export (tidak dinamis)
❌ Tidak bisa real-time collab
❌ Tidak bisa sync dengan code
❌ Tidak bisa team sharing
```

### Figma + Tokens Plugin
```
Keuntungan:
✅ Otomatis seperti PlantUML
✅ 50+ design tokens (2.5x lebih banyak!)
✅ Dynamic & real-time collaboration
✅ GitHub sync (version control)
✅ Export to CSS/JS (sync dengan code!)
✅ Team library sharing
✅ Publish library (auto-update everyone)
✅ Export to Figma (no loss of fidelity)

Kelemahan:
❌ Perlu install plugin (sekali saja)
❌ Learning curve sedikit lebih tinggi (tapi mudah)
```

**Pemenang**: Figma Tokens ✅ (lebih powerful + flexible!)
