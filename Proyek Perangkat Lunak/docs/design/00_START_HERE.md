## 💡 Pro Tips

### Tip 1: Mulai dari QUICK_START
Jangan langsung buka IMPORT_GUIDE jika belum familiar. Mulai dari QUICK_START dulu.

### Tip 2: Copy-paste siap pakai
Semua color hex, spacing, shadows sudah siap di-copy. Tinggal paste ke Figma.

### Tip 3: Jangan manual ketik
Hindari manually ketik hex values. Copy-paste dari file untuk menghindari typo.

### Tip 4: Gunakan auto layout
Figma auto layout (Shift+A) membuat components responsive otomatis.

### Tip 5: Component > Duplicate
Gunakan components (reusable) bukan duplicate (static).

### Tip 6: Backup berkala
Figma auto-save, tapi tetap "Save local file" setiap jam untuk safety.
