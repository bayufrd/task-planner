## 🎯 QUICK START (10 MENIT)

### LANGKAH 1: Install Plugin (2 menit)
```
1. Figma.com → Dashboard
2. Resources → Community → Plugins
3. Search: "Figma Tokens"
4. Install
5. ✅ Done
```

### LANGKAH 2: Buat File (1 menit)
```
1. New file
2. Name: "Task Planner - Design System"
3. Create
4. ✅ Done
```

### LANGKAH 3: Open Plugin (1 menit)
```
1. Toolbar → Plugins icon
2. Figma Tokens
3. Plugin window terbuka
4. ✅ Done
```

### LANGKAH 4: Copy-Paste JSON (2 menit)
```
1. Open: docs/design/FIGMA_TOKENS.json
2. Select All (Ctrl+A)
3. Copy (Ctrl+C)
4. Figma Tokens plugin → text area
5. Paste (Ctrl+V)
6. ✅ Done
```

### LANGKAH 5: Apply Tokens (1 menit)
```
1. Klik "Apply Tokens" button
2. Tunggu processing (1-2 detik)
3. Lihat Assets panel → Colors muncul!
4. Lihat Assets panel → Typography muncul!
5. ✅ SELESAI! 50+ tokens ready!
```

### LANGKAH 6: Verify (2 menit)
```
1. Buat rectangle test
2. Panel kanan → Fill → pilih color dari Assets
3. Color token di-apply!
4. Buat text test
5. Apply typography dari Assets
6. ✅ Semua berfungsi!
```

## 🎯 PERBANDINGAN: Draw.io PlantUML vs. Figma Tokens

### Draw.io PlantUML:
```
Keuntungan:
✅ Built-in, tidak perlu plugin
✅ Cepat untuk wireframe sederhana

Kelemahan:
❌ Limited customization
❌ Static export
❌ Tidak bisa real-time collab
❌ Design tokens terbatas
```

### Figma Tokens Plugin:
```
Keuntungan:
✅ Otomatis import (mirip PlantUML)
✅ Highly customizable
✅ 50+ design tokens (vs. PlantUML ~20)
✅ Real-time collaboration
✅ GitHub sync (version control!)
✅ Export ke code (CSS variables, JS)
✅ Publish library (team sharing)

Kelemahan:
❌ Perlu install plugin (sekali saja)
❌ Learning curve lebih tinggi (tapi mudah)
```

**Winner**: Figma Tokens ✅ (lebih powerful!)
