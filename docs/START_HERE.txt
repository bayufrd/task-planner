│  📍 Priority Scheduling Algorithm                                               │
│  ────                                                                           │
│  Automatically ranks tasks based on:                                            │
│  • Deadline urgency (days until deadline)                                       │
│  • Priority level (HIGH/MEDIUM/LOW)                                             │
│  • Reminder timing                                                              │
│  • Estimated duration                                                           │
│                                                                                 │
